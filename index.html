云计算技术 课程实验报告

学号：202000130075 姓名：朱环 班级：20.4

实验题目：利用云平台搭建个人博客

实验目的：熟悉个人博客系统的搭建。
具体包括：
参考方案：注册Github账号，搭建Hexo环境并实现个人博客搭建，撰写实验报告。

实验步骤与内容：
1.注册GitHub账号

2.搭建Hexo环境并实现个人博客搭建

2.1安装Git

2.2 安装Node.js

2.3安装Hexo

2.4在GitHub创建个人仓库

2.5生成SSH添加到GitHub

2.6 将hexo部署到GitHub

结论分析与体会：

  1. 在使用hexo搭建个人博客时，每次更新要先清除之前的，在重新生成网站，上传到GitHub上
  2. Hexo是快速、简洁且高效的博客框架，可以理解为搭建博客的模型。通过运用模型，省去了自己搭建博客的麻烦。
  3. Hexo常见命令

hexo new “postName” #新建文章

hexo new page “pageName” #新建页面

hexo generate #生成静态页面至public目录

hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）

hexo deploy #部署到GitHub

hexo help # 查看帮助

hexo version #查看Hexo的版本

也可以用缩写

hexo n #新建文章

hexo g #生成静态页面至public目录

hexo s #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）

hexo d #部署到GitHub
